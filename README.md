# Sapper

***
## Приложение:
Популярная во всём мире игра "Саппёр". Выбери количество клеток и играй! 
***
## Стек:
  - React
  - Typescript
  - SCSS module
  - Tailwind CSS
  - Vite
***
## Скрипты:
1. Для скачивания зависимостей:
```bash
npm install
```

2. Для запуска в режиме разработки:
```bash
npm start
```

3. Для сборки:
```bash
npm run build
```

4. Для запуска собранной версии(после npm run build):
```bash
npm run preview
```
